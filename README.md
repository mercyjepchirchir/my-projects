# PraRoz Web Design Landing Page

## Overview
PraRoz is a modern frontend landing page built with HTML and CSS. The website features a navigation menu, search bar, hero section, login form, and social media integration.

## Features
- Responsive navigation menu
- Search bar
- Hero section with call-to-action button
- Login form interface
- Social media icons
- Modern background image design
- Custom CSS styling

## Technologies Used
- HTML5
- CSS3
- Ionicons

## Project Structure
```
project/
├── index.html
├── style.css
└── images/
```

## How to Run
1. Clone or download this repository.
2. Open `index.html` in any modern web browser.

## Learning Objectives
This project was created to practice:
- HTML page structure
- CSS styling and layout
- Navigation bar design
- Form design
- Responsive web development concepts
